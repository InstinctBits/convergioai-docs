## Prerequisites

| Requirement | Minimum version |
| ----------- | --------------- |
| Node.js     | 20.0+           |
| PostgreSQL  | 15+             |
| npm         | 9.0+            |
| n8n         | 1.0+ (optional) |

You'll also need:

- An **Anthropic Claude API key** for AI features
- An **email account** with IMAP/SMTP access
- A **Cal.com account** (optional, for scheduling)
