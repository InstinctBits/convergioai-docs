# CampaignBoost

!!! warning "Coming Soon"
    CampaignBoost is currently in development. The UI framework is in place, with full functionality rolling out in a future release.

CampaignBoost is Convergio AI's email campaign module. It provides tools for creating, managing, and analyzing email marketing campaigns with contact list management and reusable templates.

## Planned features

### Compose

Create email campaigns with a visual editor:

- Rich text editor for email content
- AI-assisted copywriting
- Merge tags for personalization
- Preview across devices

### Contacts

Manage your contact lists:

- Import contacts from CSV
- Segment contacts by tags and attributes
- Opt-in/opt-out management
- Contact activity history

### Campaigns

Track sent and scheduled campaigns:

- Campaign status (draft, scheduled, sending, sent)
- Delivery metrics
- A/B testing support
- Send scheduling with timezone optimization

### Templates

Build reusable email templates:

- Pre-built template gallery
- Custom HTML templates
- Brand-consistent design system
- Template versioning

### Analytics

Measure campaign performance:

- Open rates and click-through rates
- Bounce and unsubscribe tracking
- Conversion attribution
- Campaign comparison
