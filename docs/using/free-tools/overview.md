# Free Tools

Convergio AI includes a set of free tools available to all users — no subscription required. These tools are accessible from the dashboard at `/dashboard/free-tools`.

## AI Prompt Generator

The Prompt Generator helps you craft effective AI prompts using proven frameworks. Select a template, customize the parameters, and generate a ready-to-use prompt in seconds.

### Available frameworks

| Framework | Best for | Description |
| --------- | -------- | ----------- |
| **Standard Prompt** | General use | Clear, direct prompt with context and constraints |
| **Role-Based Prompt** | Expert responses | Assigns a specific role/persona to the AI |
| **Step-by-Step Prompt** | Complex tasks | Breaks the task into sequential steps |
| **Expert System Prompt** | Domain-specific | Configures the AI as a domain expert with specialized knowledge |
| **Constraint-Based Prompt** | Controlled output | Defines strict boundaries and output format |

### How to use

1. Navigate to **Free Tools** in the dashboard sidebar
2. Select a prompt framework from the template list
3. Fill in the customization fields (topic, context, constraints)
4. Click **Generate** to create your prompt
5. Click **Copy** to copy the generated prompt to your clipboard

### Example: Role-Based Prompt

```
You are a senior marketing strategist with 15 years of experience
in B2B SaaS. Your task is to create a content calendar for Q2 2026
that focuses on product-led growth for a CRM platform.

Consider: target audience (SMB founders), content channels
(LinkedIn, blog, email), and budget constraints ($5K/month).
```

!!! tip "Prompt quality"
    The framework templates are designed around best practices for modern AI models. Using structured prompts consistently produces better results than freeform instructions.
